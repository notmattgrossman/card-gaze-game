# Card Matching Game

A memory matching game built with p5.js. Match pairs of cards by hovering over them.

## How to Play

- Hover over a card for 5 seconds to flip it
- Flip two cards to find matching pairs
- Matched pairs stay flipped; mismatched cards flip back after 1.5 seconds
- Complete the game by matching all 6 pairs

## Setup

Open `index.html` in a web browser.
